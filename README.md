## 简介 (Introduction)
- DogeMediaPlayer 是一款基于 Qt 框架开发的简单视频播放器，是我在IMNU期间(2023)完成的一个实训作业。

## 依赖 (Prerequisites)
- 🚨 必须安装：[K-Lite Codec Pack](https://codecguide.com/download_k-lite_codec_pack_basic.htm)。
- 我使用了Qt 5.14.2的MinGW 64-bit构建包，还需要Qt Multimedia模块。

## 功能特性 (Features)
- 视频播放与暂停
- 网络MP4播放
- 进度条拖动跳转
- 全屏播放
- 音量控制
- 快捷键
